# UI/SEO Optim Transformer (`prospektweb.uiseooptimt`)

Bitrix-модуль для управляемых UI/SEO доработок Aspro Premier и безопасного управления свойствами инфоблоков.

## Возможности
- Проверка зависимостей (`iblock`, `catalog`, `sale`) при установке.
- Идемпотентное создание/обновление свойств товаров/ТП через сервисный слой.
- Свойство `TR_CASE` («Склонение наименования», множественная строка) для инфоблока товаров.
- Безопасный patch Aspro-шаблонов с backup + checksum/hash-контролем.
- Настройки модуля через `Option` с typed-обёрткой.
- Удаление с шагом подтверждения и флагом `remove_data`.

## Структура
- `install/*` — установка/удаление модуля.
- `lib/Service/*` — сервисный слой.
- `options.php` — admin UI для настроек.
- `include.php` — автозагрузка классов.
- `assets/*` — ресурсы модуля.

## Установка
1. Поместите папку модуля в `/local/modules/prospektweb.uiseooptimt`.
2. В панели Bitrix: **Marketplace → Установленные решения → Установить**.
3. На шаге установки модуль попробует автоматически определить инфоблок товаров и SKU; при необходимости выберите ID вручную.

## Удаление
1. В панели Bitrix: **Marketplace → Установленные решения → Удалить**.
2. На шаге удаления выберите:
   - `N` — оставить данные (в т.ч. `TR_CASE`).
   - `Y` — удалить только свойства, созданные модулем.
3. Модуль восстановит оригинальные Aspro-файлы из backup.

## Настройки
В `Настройки → Настройки продукта → Настройки модулей → UI/SEO Optim Transformer`:
- `ENABLED`
- `PRODUCTS_IBLOCK_ID`
- `OFFERS_IBLOCK_ID`

## Надёжность
- PHP 8+
- Повторная установка безопасна (re-run idempotency).
- Ошибки установки/удаления логируются и показываются администратору.
